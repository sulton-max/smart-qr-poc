# v0.1 — Product foundation (guest-first)

*Last updated: 2026-06-18*

**Status:** ✅ Complete · **Started:** 2026-06-03 · **Completed:** 2026-06-18

### Iteration 1 — Generation and serving: create a code and have it resolve end to end

- [x] Ability to generate a QR code.
- [x] Ability to download a QR code as SVG and PNG.
- [x] Ability to route a scan to its destination with rules and a fallback.

### Iteration 2 — Identity: a code has an owner without sign-in

- [x] Ability to log in as a guest.
- [x] Ability to own codes that only you can manage.

### Iteration 3 — Manage codes: an edit takes effect on the next scan

- [x] Ability to edit, search, enable/disable, and delete your codes.
- [x] Ability to have edits take effect on the next scan.

### Iteration 4 — Verification: confirm v0.1 works end to end

- [x] Create a QR code in the builder → it renders and downloads as SVG and PNG.
- [x] Open the code (scan or its short link) → resolves to the destination; no rule match → the fallback.
- [x] As a fresh guest, create a code → it's yours; another guest can't see or manage it.
- [x] Edit the destination → the next open resolves to the new one.
- [x] Disable a code → it stops resolving; enable it → resolves again.
- [x] Search your codes → returns the matches.
